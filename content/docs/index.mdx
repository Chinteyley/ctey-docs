## Explore My Work

Welcome to my documentation site. I am Chintey Ley, a software developer specializing in TypeScript, React, React Native, and Python. I build full-stack web applications, cross-platform mobile apps, and machine learning tools. My work focuses on solving real problems with clean, maintainable code and thoughtful user experiences.

This site serves as a central place to document my projects, the decisions behind them, and the technologies that power them. I created it both as a reference for anyone interested in my work and as a way to share what I have learned along the way.

## What You Will Find Here

This documentation covers several areas of my work:

- **Project documentation** -- Detailed write-ups for each project, including architecture decisions, features, and development process. Each project page explains what the project does, why I built it, and how it works under the hood.
- **Tech stacks** -- Breakdowns of the technologies used in each project. My stack typically includes Next.js, Angular, React Native with Expo, Flutter, PyTorch, and TensorFlow, among others.
- **Live demos and links** -- Where available, links to live deployments, app store listings, and source code repositories so you can try things out for yourself.
- **Contact information** -- Ways to reach me for collaboration, questions, or just to connect.

## Technologies

Across my projects, I work with a range of modern tools and frameworks. On the frontend, I primarily use React and Next.js for web applications and React Native with Expo for mobile. On the backend, I work with Node.js, Python, and various database systems. For machine learning projects, I use PyTorch and TensorFlow for model training and inference. I also have experience with Angular, Flutter, and cloud deployment platforms.
