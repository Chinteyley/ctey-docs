## Who I Am

I am Chintey Ley, a software developer with a passion for building practical, user-focused applications. My work spans web development, mobile apps, and machine learning, with a focus on creating tools that solve real problems.

## My Background

I specialize in modern web technologies including TypeScript, React, and Next.js for frontend development, and Node.js, Flask, and Supabase for backend systems. On the mobile side, I build cross-platform applications using React Native with Expo and Flutter. My machine learning projects use Python with PyTorch and TensorFlow for computer vision and generative AI.

## What I Do

My projects range from production-grade platforms like Jomnouy, an AI-powered exam preparation tool for Cambodian students, to creative experiments like an Anime Face GAN that generates character images using deep learning. I enjoy working across the full stack, from designing clean user interfaces to building reliable backend services and deploying with Docker, NGINX, and cloud platforms.

## Technical Skills

- **Frontend**: TypeScript, React, Next.js, Tailwind CSS, ShadCN, Motion
- **Mobile**: React Native, Expo, Flutter, Dart
- **Backend**: Node.js, Express, Flask, Supabase, PostgreSQL, MongoDB
- **AI/ML**: Python, PyTorch, TensorFlow, Keras, OpenCV, Gradio
- **DevOps**: Docker, NGINX, Vercel, GitHub Actions
