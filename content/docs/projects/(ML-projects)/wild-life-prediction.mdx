## Overview

African Wildlife Classifier is an AI-powered image classifier that excels in identifying African wildlife species with 95.6% accuracy. It employs a fine-tuned AlexNet deep learning model to classify four iconic African safari animals with real-time confidence scores.

### Features

- Identifies 4 African wildlife species: Buffalo, Elephant, Rhino, Zebra
- Fine-tuned AlexNet architecture with 95.6% validation accuracy
- Real-time image classification with confidence scores
- Interactive web interface for wildlife enthusiasts
- Data augmentation and preprocessing pipeline
- Educational tool for wildlife research

### Tech Stack

- PyTorch
- Gradio
- AlexNet
- Computer Vision
- Deep Learning
- Python
