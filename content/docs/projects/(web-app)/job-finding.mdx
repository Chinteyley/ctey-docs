## Overview
A job finding web application that helps students search for job opportunities, view job details, and apply for positions directly through the platform. Includes filters for location, job type, and more.

### Key Features
- Job search and filtering
- View job details
- Apply for jobs online
- User-friendly interface

### Tech Stack
- Next.js
- Tailwind CSS
- ShadCN
- Motion
- Supabase
