## Overview
A countdown web application that helps students keep track of their important exam dates. It features a clean and intuitive design with a countdown timer displaying the remaining time until the exam.

### Key Features
- Countdown timer for exams
- Clean and intuitive UI
- Responsive design

### Tech Stack
- Next.js
- Tailwind CSS
- ShadCN
- Motion
