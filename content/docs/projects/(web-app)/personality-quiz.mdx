## Overview
An interactive personality quiz application that helps users discover their elemental personality type (Fire, Water, Earth, or Air) through a series of engaging questions. The app provides personalized results with detailed descriptions and visual elements.

### Key Features
- Interactive 5-question personality assessment
- Four elemental personality types: Fire, Water, Earth, and Air
- Dynamic progress tracking
- Personalized results with detailed descriptions
- Random pet images fetched from Dog API and Cat API
- Modern, responsive UI with smooth animations
- Restart functionality to retake the quiz

### Tech Stack
- React 19
- TypeScript
- Vite (with Rolldown)
- Tailwind CSS
- React Router
- Lucide React (icons)
- ShadCN UI components
- Class Variance Authority
