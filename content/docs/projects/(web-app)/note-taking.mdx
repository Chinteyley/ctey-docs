## Overview
A note-taking web application that allows users to manage and organize their personal notes. Features a clean UI for creating, editing, deleting, and categorizing notes.

### Key Features
- Create, edit, and delete notes
- Categorize notes
- Clean and intuitive UI

### Tech Stack
- Angular
- MongoDB
- Express
- Node.js
- Docker
- NGINX
- OpenAI API
