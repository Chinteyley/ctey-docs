## Overview
A web application that utilizes cutting-edge technology to accurately and efficiently record attendance. It processes data securely and integrates with a dashboard for managing attendance records. Notifications are sent to update individuals on their attendance status.

### Key Features
- Automatic attendance detection and recording
- Secure data processing
- Dashboard for attendance management
- Real-time notifications

### Tech Stack
- Next.js
- Flask
- Prism
- PostgreSQL
