I have worked on several projects that showcase my skills and expertise in web and mobile development, machine learning, and interactive applications. These projects range from full-stack web platforms and mobile apps to AI-powered tools and fun browser-based games. Click on a project below to learn more about its features, tech stack, and development process.

## Project Categories

My projects generally fall into four categories, each reflecting a different area of my development experience.

**Web Applications** -- These are full-stack projects built with frameworks like Next.js and Angular. They include platforms for job searching, note-taking, attendance management, and educational tools. Most of these projects use server-side rendering, REST APIs, and modern frontend patterns to deliver responsive and accessible user interfaces.

**Mobile Apps** -- Cross-platform mobile applications built with React Native and Expo or Flutter. These apps target both iOS and Android from a single codebase. My mobile work includes SnapPosts, a poster generation app that combines camera capture with AI-powered design, as well as other utility apps focused on practical everyday use.

**Machine Learning** -- AI-powered tools and experiments using PyTorch, TensorFlow, and computer vision techniques. These projects explore generative models, image classification, and predictive analytics. They include work with generative adversarial networks for image synthesis and wildlife species classification from photo data.

**macOS Apps** -- Native macOS applications built with Swift and SwiftUI. These are lightweight, focused utilities designed for the Mac desktop experience.
